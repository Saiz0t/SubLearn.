# SubLearn.
Проект: Sublearn
Концепция: Десктопное приложение для изучения иностранного языка на основе контекста любимых фильмов и сериалов.
Основные функции:
• Умный парсинг: Автоматическое извлечение английских слов из файлов субтитров формата .srt.
• Фильтрация: Очистка текста от таймкодов, тегов и стоп-слов (выделение только значимых лексем от 4 букв).
• Автоматический перевод: Интеграция с API переводчиков для мгновенного получения значений слов.
• Интерактивное обучение: Режим викторины (теста) с вариантами ответов для закрепления выученного материала.
Технологический стек:
• Язык: Python
• Интерфейс: CustomTkinter (современный графический интерфейс в стиле Material Design/Dark Mode).
• Библиотеки: translate (перевод), re (регулярные выражения для обработки текста), threading (многопоточность для плавной работы UI во время запросов к сети).
Как это работает:
Пользователь загружает файл субтитров, программа анализирует лексику, переводит её и формирует локальную базу знаний. На основе этой базы создаются интерактивные карточки, которые помогают запомнить слова, встреченные в конкретных сценах фильма.
